# ZyboZ7-20-Ultrasonic-Ruler
Uses an HC-SR04 Ultrasonic senor along with the ZyboZ7-20 to display a single digit distance from the device. With cm or inches being selectable

#Pins

T14, T15, P14, R14, V12, W16, J15 - Are all used for the 2 digit seven segment display \n
V15 - HC-SR04 Echo pin \n
W15 - HC-SR04 Trig pin
