# ZyboZ7-20-Ultrasonic-Ruler
<br>
Uses an HC-SR04 Ultrasonic senor along with the ZyboZ7-20 to display a single digit distance from the device.<br> With cm or inches being selectable.
<br>
#Pins
<br>
T14, T15, P14, R14, V12, W16, J15 - Are all used for the 2 digit seven segment display <br>
V15 - HC-SR04 Echo pin <br>
W15 - HC-SR04 Trig pin
